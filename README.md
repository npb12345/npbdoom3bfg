# NPBDoom3BFG

NPBDoom3BFG is a personal modification of the RBDoom3BFG source port of **Doom 3 BFG Edition**.

This project contains gameplay, visual, and engine tweaks while experimenting with the Doom 3 BFG source code.

Examples of modifications include:

- Weapon behavior tweaks
- Laser sight implementation
- Crosshair improvements
- Gameplay experimentation
- Visual and lighting adjustments

## Base Projects
